Software Engineering — Game Project

This project was part of the Software Engineering course at Nipissing University. The purpose of this project was to apply proper software engineering practices throughout the development cycle for a game. This included designing the complete application while creating the necessary documents and performing thorough tests for all added features. This project was done in a team of six classmates.

The game was developed with Unity3D. The directive was to create a serious game that is related to Games for Health. In particular, should development continue with this project, the goal was for it to be used to educate the user about asthma. As the project had a short window for development, the game is a prototype. The focus was to develop the user interface, controls, and mechanics, rather than on level design. Please keep this in mind if you choose to check out the playable demo below!

While all team members contributed to the documentation, design, and testing, the development portion was split up. My main contribution was to design and develop the HUD and integrate it with the game mechanics, as well as apply the animations of the Unity Store assets used so they would syncronize well with user input.
