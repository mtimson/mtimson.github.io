Virtual Vimy Ridge Exhibit

This project is a collaboration between the Military Communications and Electronics Museum in Kingston, ON and Nipissing University. It is a virtual museum exhibit that is being developed to commemorate the 100 year anniversary of the Battle of Vimy Ridge. It was developed using Cesium, an open source JavaScript library for 3D globes.

My contributions include developing the main application along with my colleagues, and developing tools for additional data, models, documents, and more to be added to the system. The features of the application are illustrated below. More details on this project are provided here.

  • Role: Development
  • Organization: Nipissing University
  • Year: 2015-2016
  • Application overview: illustrates the User Interface and maps draped over terrain.
  • 3D "barrage lines" constructed from map; view documents by interacting with file browser
  • Generated trenches with textures; view documents by interacting with models
  • Select multiple maps to display on globe; change the order in which they are layered; individually manipulate their transparencies
