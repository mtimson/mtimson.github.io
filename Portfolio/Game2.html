Game AI Project

This project was part of the Advanced Game Design and Development course at Nipissing University. The focus was to build a new game based in part on our project from the previous semester. With the new project, the focus would be on developing game AI, as that made up the majority of the course instruction. My group decided to make a different type of game from the previous project, specifically a shooter. This allowed us to create a variety of enemy types which all act as autonomous agents, exhibiting different steering behaviours. The goal of the game is the same (i.e. education about asthma upon further development), but in this game the player is defending a lung from asthma triggers. The game was developed as a stand alone, so a playable version can be downloaded here.
